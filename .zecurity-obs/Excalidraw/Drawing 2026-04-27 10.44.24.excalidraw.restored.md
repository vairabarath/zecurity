# ACL Sync — System Architecture

This page is the visual companion to [`docs/acl-sync-architecture.md`](../../docs/acl-sync-architecture.md). The Excalidraw scene below shows the three ACL delivery paths between Controller, Connector, and Client. Mermaid sequence diagrams further down render natively in Obsidian preview.

**Color legend**

- 🟢 **green** — control plane (gRPC stream, signed ACL push)
- 🔵 **blue** — data plane (mTLS tunnel from Client through Connector to internal resource)
- 🔴 **red** — sync recovery path (401 from Connector + HTTPS resync from Controller)

---

## Sequence: Login + Initial ACL

```mermaid
sequenceDiagram
    autonumber
    actor U as User
    participant CLI as Client
    participant CTL as Controller
    U->>CLI: zecurity login
    CLI->>CTL: OIDC + PKCE auth
    CTL->>CTL: compile + HMAC-sign ACL (v=N)
    CTL-->>CLI: tokens + signed_acl{v=N}
    CLI->>CLI: verify, cache in memory
    Note over CLI: split-tunnel routing active
```

## Sequence: Revocation (admin removes access)

```mermaid
sequenceDiagram
    autonumber
    actor Adm as Admin
    participant CTL as Controller
    participant CON as Connector
    participant CLI as Client
    Adm->>CTL: remove user from group
    CTL->>CTL: recompile ACL → v=N+1
    CTL-->>CON: push v=N+1 (gRPC stream, instant)
    CON->>CON: PolicyCache := v=N+1
    Note over CLI: still cached v=N
    CLI->>CON: tunnel to revoked resource
    CON-->>CLI: 401 policy_changed (v=N+1)
    CLI->>CTL: GET /api/v1/acl
    CTL-->>CLI: 200 signed_acl v=N+1
    CLI->>CLI: atomic swap → v=N+1
```

## Sequence: 30-second Background Poll

```mermaid
sequenceDiagram
    autonumber
    participant CLI as Client
    participant CTL as Controller
    loop every 30 s
        CLI->>CTL: GET /api/v1/acl<br/>If-None-Match: W/"v-N"
        alt unchanged
            CTL-->>CLI: 304 Not Modified
        else changed
            CTL-->>CLI: 200 signed_acl v=M (ETag W/"v-M")
            CLI->>CLI: verify, atomic swap
        end
    end
```

```mermaid

    Admin[Admin UI] -->|GraphQL mutation| CTL
    CTL[(Controller<br/>Go · gRPC + HTTPS)]

    RES[(Internal Resource)]

    CTL ===|gRPC stream<br/>ACL push v=N| CON
    CTL -.->|HTTPS<br/>signed ACL on login + 30s poll| CLI
    CLI ==>|mTLS tunnel<br/>SPIFFE auth| CON
    CON -->|forward TCP| RES
    CON -. 401 policy_changed<br/>→ triggers HTTPS resync .-> CLI
```
